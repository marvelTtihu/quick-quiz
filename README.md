# Quick Quiz

This is a quick quiz that contains basic front-end questions, and only has 5 questions.

## Features 

- A basic HTML5/CSS3/Javascript questions.
- Responsive Design (Mobile/tablet).
- Hover and active button effects.

## How to run

- Clone this repo and locate ``quiz-app`` files.
- Open ``index.html`` in your web browser.  
- The quiz will start when the page loads.
